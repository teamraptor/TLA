# Raptor compiler

Special assignment for Theory of Languajes and Automatas course at [ITBA](https://www.itba.edu.ar)

To compile.. the compiler:

```
ant clean jar
```

To compile a file `ejemplo.rawr`:

```
./compile.sh ejemplo
```

To execute the file:

```
java ejemplo
```
